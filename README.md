# desired-database-WebApp

This repository shows a database system designed to store and process telemetry data from electric motorcycles used in Student Research Group [E-Moto AGH](https://e-moto.agh.edu.pl) and is a part of [KN Spectrum](https://spectrum.agh.edu.pl) contribution to the project.

### Tech Stack

- **PostgreSQL** - main database engine
  - **[TimescaleDB](https://www.tigerdata.com)** - database extension for efficient processing of timeseries data
  - **[PostGIS](https://postgis.net)** - database extension for world coordinates handling and visualization
- **dbmate** - lightweight migrator tool
- **Docker** - for easy and robust setup
- **K8S** - coming soon...
